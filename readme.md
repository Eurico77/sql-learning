## o que é sql

- S: strutctured
- Q: query
- L: language

## as partes do sql

- DDL
  - Data Definition Language

  ```sql
  CREATE TABLE
  ALTER TABLE
  DROP TABLE
  ```

  - definição dos dados para persistência, atributos e etc...
- DML
  - Data Manipulation Language

  ```sql
  INSERT INTO
  UPDATE
  DELETE
  WHERE
  ```

  - manipular e modificar o banco
- DQL
  - Data Query Language

  ```sql
  SELECT 
  WHERE

  ```

- DCL
  - Data Control Language

  ```sql
  GRANT
  REVOKE

  ```
